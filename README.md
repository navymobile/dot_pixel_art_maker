# ドット絵共有アプリ（仮称）

## 概要

会って受け取り、出自が残る16×16ドット絵共有アプリ。

ドット絵は「作品」であり、
genは「人と人のつながりの長さ」を表す。

SNSではない。
評価文化もない。
対面交換を前提とする。

---

## コアコンセプト

- 16×16固定
- 対面QR交換
- 出自の追跡可能性
- 最小構成

---

## 技術スタック

- Flutter
- CustomPainter（描画）
- Riverpod（状態管理予定）
- QR生成・読み取り

---

## データ構造（概要）

Dot:

- dna (UUID)
- creator_id
- pixels (256配列)
- gen
- received_from

---

## MVP範囲

- ドット描画
- データ化
- QR交換
- gen増加
